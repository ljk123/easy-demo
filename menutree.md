# 目录结构


>```
>├──app 应用目录
>│   ├─config //配置文件目录
>│   │   ├─cache.php //缓存配置文件
>│   │   ├─cache.php //公共配置文件
>│   │   ├─database.php //数据库配置文件
>│   │   ├─log.php //日志配置文件
>│   │   ├─redis.php //reids配置文件
>│   │   ├─swoole.php //swoole环境配置文件
>│   ├─controller //控制器目录
>│   ├─... //其他自定义目录
>├──runtime 运行时目录
>├──web web目录
>│   ├───index.php //项目入口文件
>└─-swoole //swoole环境入口文件
>```
